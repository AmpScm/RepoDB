Features
--------

* Asynchronous Operations
* Batch Operations
* Bulk Operations
* Caching
* Connection Persistency
* Database Helpers
* Database Settings
* Expression Trees
* Extension Methods
* Field Mapping
* Inline Hints
* Massive Operations (Generics/Explicits/MethodCalls/TableBased)
* Multi-Resultset Query
* Query Builder
* Repositories
* Resolvers (CLR Types, DB Types)
* Statement Builder
* Tracing
* Transaction
* Type Mapping
